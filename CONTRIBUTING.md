# Contributing to StockKit

Thank you for your interest in contributing to StockKit!

## Ways to Contribute

### Report Issues
- Found a bug? Open an issue with steps to reproduce
- Have a feature idea? We'd love to hear it

### Share StockKit
- Star this repo ⭐
- Share [stockkit.net](https://stockkit.net) with fellow traders
- Write about your experience using StockKit

### Code Contributions
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Community

- Website: [stockkit.net](https://stockkit.net)
- Subscribe: [Free AI Reports](https://stockkit.net)

## Code of Conduct

Be respectful, constructive, and inclusive. We're all here to build better tools for traders.
